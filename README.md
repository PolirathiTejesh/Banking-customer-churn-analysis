# Banking Customer Churn & Retention Analysis

## Project Overview

This project analyzes banking customer churn using Power BI to identify customer behavior, retention trends, and business improvement opportunities. The dashboard provides interactive visualizations and key performance indicators (KPIs) to support data-driven decision-making.

## Tools Used

- Microsoft Excel
- Power BI
- Power Query
- DAX

## Key KPIs

- Total Customers
- Churned Customers
- Churn Rate
- Retained Customers
- Active Customers
- Active Customer Rate
- Average Satisfaction Score

## Key Business Insights

- Active customers have higher retention rates.
- Customers with fewer products are more likely to churn.
- Complaint history is associated with increased churn.
- Most customers belong to the 26-45 age group.
- Lower satisfaction scores are linked to higher churn.

## Business Recommendations

- Improve engagement with inactive customers.
- Promote additional banking products.
- Strengthen complaint resolution processes.
- Improve customer satisfaction initiatives.
- Monitor high-risk customers proactively.
